# Future Shivang’s Toolkit  
A two-page Streamlit application designed to help Future Shivang show up more often in moments that matter.

## Tools
- **Before the Moment** – Prepare for conversations, meetings, interviews.
- **After the Moment** – Reflect on what actually happened and who showed up (Past/Present/Future Me).

## Run Locally
#pip install -r requirements.txt
#streamlit run Home.py


## Structure
- Home.py – Landing page  
- pages/ – Before & After pages  
- .streamlit/config.toml – Theme configuration  

Built for MGTA 497 / Managerial Decision-Making Final Project.
